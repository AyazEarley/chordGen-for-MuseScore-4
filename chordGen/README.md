# chordGen
MuseScore 4 plugin that generates diatonic harmonies in Major/Minor Keys.
Uses Markov changes to adhere to typical tonic/dominant rules.

## Install
Move the folder into `Program Files/MuseScore 4/Plugins`. 

In MuseScore 4, navigate to `Plugins` -> `Manage Plugins` click on the plugin and click enable.

## Usage
Navigate `Plugins` -> `Chord Generator Plugin` 

Select a measure on the score, select a key, mode, duration, and length, click generate. Ensure 
that the key signature matches the key you are generating in, or else you may generate chords 
with strange accidentals.
